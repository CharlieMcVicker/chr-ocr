# Goal Description

The goal of Task-9 is to extract the hand-labeled data from the custom annotation server and format it into a standard ground-truth dataset that can be used for downstream Tesseract fine-tuning and data augmentation. We also need to acquire the base `chr.traineddata` model.

## Proposed Changes

We will create a Python script that parses the `training_data/manifest.json` file generated by your custom annotation server, filters for completed labels, and exports them to a clean dataset directory.

### Scripts

#### [NEW] [extract_ground_truth.py](file:///Users/charlesmcvicker/code/phoenix/scripts/extract_ground_truth.py)
This script will:
1. Load `training_data/manifest.json`.
2. Filter for items where `"status" == "labeled"`.
3. Create a clean `dataset/ground-truth/` directory.
4. For each labeled item:
   - Copy the original image from `training_data/<image_path>` to `dataset/ground-truth/`.
   - Create a corresponding text file (e.g., `image_name.gt.txt`) containing the Cherokee text from the `"label"` field.
5. Provide a summary count of how many labeled examples were successfully extracted.

#### [NEW] [download_base_model.sh](file:///Users/charlesmcvicker/code/phoenix/scripts/download_base_model.sh)
A simple bash script to download the official `chr.traineddata` from the `tesseract-ocr/tessdata_best` repository into a `dataset/model/` directory, which we will use as the starting point for fine-tuning.
